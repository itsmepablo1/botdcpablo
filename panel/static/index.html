🔊 Auto Voice Channel

⚙️ Konfigurasi Hub

Member yang join channel ini akan otomatis dibuatkan VC baru.

📖 Cara Setup Cepat

1 Di Discord, jalankan /setupvoice create
2 Bot otomatis buat kategori 🔊 Voice Hub + hub channel
3 Member join hub → VC baru dibuat otomatis untuknya
4 VC kosong → dihapus otomatis oleh bot
5 Atau set Channel ID manual di form kiri lalu klik Simpan

📈 Status Channel Config

⚙️ Konfigurasi

Channel yang tampilkan total member. Gunakan /status setup untuk buat otomatis.

ℹ️ Cara Setup Cepat

1 Di Discord, jalankan /status setup
2 Bot otomatis buat kategori + 2 channel VC
3 Channel update realtime setiap ada perubahan member
4 Force update dengan /status refresh

🔴 Streaming Notification Config

⚙️ Konfigurasi

Channel tempat notifikasi streaming dikirim. (Opsional)
Hanya member dengan role ini yang dideteksi sebagai streamer.
Role ini otomatis ditambah saat live & dihapus saat offline.
